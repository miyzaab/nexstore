# Nex Store

Website toko furnitur kantor bekas berkualitas premium.

## Fitur
- Responsive design (mobile-friendly)
- Dark mode support
- Animasi scroll yang smooth
- Mobile navigation menu

## Halaman
- `index.html` - Beranda
- `catalog.html` - Katalog produk
- `about.html` - Tentang Kami
- `contact.html` - Hubungi Kami

## Teknologi
- HTML5
- Tailwind CSS
- JavaScript (Vanilla)

## Cara Menjalankan
Cukup buka file `index.html` di browser Anda.
